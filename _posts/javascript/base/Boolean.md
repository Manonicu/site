Boolean 有两个值 true/false。

| 数据类型  | 转换为true的值 | 转换为false的值 |
| --------- | -------------- | --------------- |
| Boolean   | true           | false           |
| String    | 任意飞空字符串 | 空字符串        |
| Number    | 任何非零值     | 0和NaN          |
| Object    | 任何对象       | null            |
| Undefined |                | undefined       |
